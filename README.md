# Diabetes-prediction
This project uses Machine Learning techniques to predict whether a person is likely to have diabetes based on various health-related factors. The model is trained on a diabetes dataset and helps in early detection by analyzing input features such as glucose level, BMI, blood pressure, age, and other medical parameters.
